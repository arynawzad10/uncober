    Load Tweaks

    This option makes Substitute load extensions that are commonly referred to as tweaks.

    Disable Auto Updates

    This option makes the jailbreak effectively disable the system's Software update mechanism to prevent the system from automatically upgrading to the latest available firmware which may not be Supported by the jailbreak at that time.

    Share Diagnostics Data

    Share anonymous OS crash logs with us so we can investigate bugs further. We only send kernel panic logs. No trackable or other personal information will be sent.

    Export Kernel Task Port

    This option makes the jailbreak modify the host-port to grant any process access to the host-priv-port. This option effectively grants any process access to the kernel task port (TFPO) and allows re-jailbreaking without exploiting again. This option is considered unsafe as the privilege this option effectively grants bad to processes can be used for purposes by malicious apps.

    (Re)Install OpenSSH

    This option makes the jailbreak (re)install the openssh package.

    Restore Root Filesystem

    This option makes the jailbreak restore the root filesystem (RootFS) to the snapshot which is created by the system when the device is restored. This option effectively allows uninstalling the jailbreak without losing any user data.
